Detailed Description

template<int N_ACTIONS, class URand = SimpleURand>
class mcts::UCTreeNode< N_ACTIONS, URand >

Represents a node in a UCT tree.

This provides the main data structure and implementation of the UCT (Upper Confidence Tree) algorithm.

Template Parameters
N_ACTIONSthe number of actions in the action domain.
URandclass used to generate uniform random numbers in range [0,1). This is used internally during selection and rollout.

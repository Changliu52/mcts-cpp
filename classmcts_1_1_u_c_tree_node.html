Detailed Description

template<int N_ACTIONS>
class mcts::UCTreeNode< N_ACTIONS >

Represents a node in a UCT tree.

This provides the main data structure and implementation of the UCT (Upper Confidence Tree) algorithm.

Template Parameters
N_ACTIONSthe number of actions in the action domain.
